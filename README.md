# SmartATS

SmartATS is a beginner-friendly AI Resume Screening and Applicant Tracking System.
It will use SBERT to compare resumes with job descriptions and explain every score
shown to HR.

## Technology

- Python 3.12
- Streamlit
- Sentence-BERT
- Matplotlib
- Dictionaries, lists and CSV files
- Pytest

## Open in VS Code

Open the `SmartATS` folder in VS Code. Setup and run instructions will be added as
the project is developed step by step.
